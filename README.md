# Tribu Camp · Tryouts

Primera versión de la aplicación para registrar las aptitudes de campistas y distribuirlos equitativamente entre 16 tribus.

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
```

La aplicación es un proyecto Vite estático compatible con Vercel. Los datos se guardan en `localStorage` del navegador; esta primera versión no requiere base de datos.
